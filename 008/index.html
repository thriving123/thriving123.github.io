IDEA2020.01版本破解教程

发表于 2020-04-19 更新于 2020-08-19 分类于 JSP学习 Valine：

首先我们下载IDEA 单击跳转下载页

然后下载破解补丁 单击跳转下载页

开始安装IDEA（具体安装过程这里不演示了）

安装完成我们运行IDEA

前面配置根据自己喜好配置 直到这里选择试用

然后重启IDEA

将破解补丁压缩包内的lib里的jetbrains-agent.jar解压出来 拖动到IDEA窗口里然后restart

选择激活方式，我这里选择的License server


重新点击注册

server address一般会自动出来

然后点击active即可破解

（如果是选择的Activation的话输入破解补丁压缩包内的ACTIVATION_CODE.txt内的文本即可）
