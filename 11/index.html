蓝桥杯-暴力破解-01

发表于 2020-08-22 分类于 蓝桥杯 阅读次数： 阅读次数： Valine：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

​ 前言：从今天开始，我将学习蓝桥杯从0到有的实战（依靠于蓝桥杯指导老师所发布的相关练习题和PDF教学资料）

​ 由于假期即将结束，那么这几天我将一天N更用以完成我对蓝桥杯训练之前遗留的问题的解答。

好的，话不多说，正题开始，咱们上菜！

​ 问题描述：

​ 美国数学家维纳(N.Wiener)智力早熟，11 岁就上了大学。他曾在 1935~1936 年应 邀来中国清华大学讲学。一次，他参加某个重要会议，年轻的脸孔引人注目。于是 有人询问他的年龄，他回答说：“我年龄的立方是个 4 位数。我年龄的 4 次方是个 6 位数。这 10 个数字正好包含了从 0 到 9 这 10 个数字，每个都恰好出现 1 次。” 请你推算一下，他当时到底有多年轻。

解题思路：

  1. 确定这位美国数学家的年龄应该是在11岁~100岁之间
  2. 确定年龄的立方是四位数
  3. 确定年龄的立方是六位数
  4. 确定这10个数字恰好出现一次

解题代码

public class test1 {
    /**
     * 1：问题描述
     * 美国数学家维纳(N.Wiener)智力早熟，11 岁就上了大学。他曾在 1935~1936 年应邀来中国清华大学讲学。
     * 一次，他参加某个重要会议，年轻的脸孔引人注目。于是 有人询问他的年龄，他回答说：“我年龄的立方是个 4 位数。
     * 我年龄的 4 次方是个 6 位数。这 10 个数字正好包含了从 0 到 9 这 10 个数字，每个都恰好出现 1 次。”
     * 请你推算一下，他当时到底有多年轻。
     * @author 不离
     * */
    public static void main(String[] args) {
        for (int i=0;i<100;i++){
            int a=(int )Math.pow(i,3);
            int b=(int )Math.pow(i,4);
            if(String.valueOf(a).length()==4 && String.valueOf(b).length()==6){
                boolean flag=true;
                for (int j=0;j<10;j++){
                    if(String.valueOf(a).indexOf(String.valueOf(j))==-1 && String.valueOf(b).indexOf(String.valueOf(j))==-1){
                        flag=false;
                    }
                }
                if (flag){
                    System.out.println(i);
                    break;
                }
            }
        }
    }
}
